1112 空氣污染概論

Chapter 9 W11_0503 （席老師）

\(\require{mhchem}\)

  • Absorbtion 吸收
    • gas->liquid
    • 要找到一個liquid
    • 在很高的溫度、在很快的氣體流速，讓濃度很低的污染物被吸收到液體
      • 需要足夠的接觸時間
    • 世界上最好的溶劑就是水，但是水是極性的溶劑，不適合用於非極性的物質
  • Adsorption 吸附
    • gas->solid

9.1 吸收

同質互溶

判斷 污染物是否適合吸收法

  • 溶劑：便宜大量方便取得：水
    • 可以改變PH值
  • 溶質：一定要親水hydrophilic

溶於水的物質常常就是用吸收法。

  • 燃煤之後的洗滌塔， 去除 \(\ce{CO2}\)
  • 半導體厰，去除 \(\ce{IPA}\)->對吸收之後的液體，再用高效率的分餾塔，去做分離

用有機溶劑做吸收？很少的情況，除非回收價值很高。

但如果回收價值很高，往往就用【吸附】

9.1.1 吸收塔的設計種類

吸收塔

  • 吸收污染物（氣體->液體
  • 先去除粒狀物
  • 設備一定會降溫
    • 也有可能專門用來做降溫

太多種類的吸收塔，每種都有不同的功效

目的都是讓氣體和液體做均匀有效的接觸【增加接觸面積、延長停留時間】

要考慮壓損、scour（積垢）

積垢

\(\ce{Ca(OH)2 + S2O -> CaSO4}\)（永久硬度）

污染防治設備：越便宜、越堅固耐用越好

  • packed column 填充塔【最常見】
    • random packing
    • structured packing
      • 較難保養【更換清潔】
    • 填充料
      • 最常見：塑膠PE
      • 陶瓷【溫度太高、酸堿度太强時】，但重量高，容易碎
      • 金屬【溫度很高很高時】價格高、重量高
  • plate column
    • 很少使用，去除效率不好，主要是用於工業應用的回收
    • 主要用在化工廠、石化廠【氣體污染物濃度特別高】，回收氣體污染物
  • miscellaneous

在氣體進入吸收塔之前，務必先把其中的粒狀物都去除，否則很快就要維護清潔吸收塔了。

9.2 吸附

gas(adsorbate) -> solid

需要一個有效的吸附劑【adsorber】有孔的材料，具有高的吸附能量，活性炭、沸石

同質互吸

也可以吸附水中的污染物，adsorbate：水中的重金屬、VOCs

非極性的物質就要找非極性的吸附材（活性炭，黑色的） 活性炭可以用來去除臭味，VOCs往往都有臭異味。

活性炭，一般都是重複再利用的，

但是如果，活性炭吸收了有害物質，就變成危害性廢棄物，依法規掩埋

油槽，油揮發，因高壓而減壓，排出來的氣，先經過活性炭吸附，再regenerate變成氣體，再冷凝，回收

9.3 生物濾床biofiltration

把VOCs作爲碳源，提供微生物的生長，要控制溫度、適度

在台灣越來越少，實體應用有難處，因爲其控制不容易，生物有不同類型

環保法規嚴格，較難符合

9.4 重點

各類設備：

  • Incineration 燃燒
  • Condensation 冷凝
  • Absorption 吸收
  • Adsorption 吸附
  • （Biofiltration 生物過濾）
